## Overview
StudyWise is a smart study planner that helps students manage time, stay consistent, and prepare better for exams.

## Problem Statement
Students often fail to manage time properly and follow inconsistent study routines, leading to poor academic performance.

## Objective
To build a study planner app that improves productivity through structured planning and reminders.

## Key Features
- Smart timetable creation  
- Task and assignment tracking  
- Study streaks and progress tracking  
- Notifications and reminders  

## Technology Used
- **Firebase Firestore** (mandatory)  
- Firebase services for real-time data and storage  

## Deliverable
- Fully functional study planner application

## Conclusion
StudyWise helps students plan better, stay consistent, and reduce last-minute exam stress.
