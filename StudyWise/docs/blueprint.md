# **App Name**: StudyWise

## Core Features:

- Smart Timetable Generation: Automatically generate a personalized study timetable based on user preferences, task deadlines, and subject priorities. Uses a scheduling algorithm tool to optimize study time allocation.
- Task and Assignment Tracking: Allow users to create, track, and manage tasks and assignments with deadlines, priority levels, and subject categories. Saves to the Firestore database.
- Study Streaks and Progress Analytics: Track study streaks and provide visual analytics on study progress, time spent, and task completion rates. Visualize trends based on the Firestore database.
- Notifications and Reminders: Send timely notifications and reminders for upcoming tasks, deadlines, and scheduled study sessions. Uses Firebase Cloud Messaging (FCM).
- User Authentication and Preferences: Secure user authentication and customizable study preferences (subject priorities, preferred study times) integrated with Firestore.
- Offline-First Support: Enable offline access to timetables, tasks, and study session data. Synchronization with Firestore database on network restore.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) for a sense of focus and productivity.
- Background color: Light cyan (#E1F5FE), creating a calm atmosphere
- Accent color: Vibrant purple (#AB47BC), to highlight important CTAs
- Body and headline font: 'Inter', sans-serif, for a clean and modern look
- Use clean, minimalistic icons to represent subjects, tasks, and study sessions.
- Prioritize a clean and intuitive layout with clear visual hierarchy and easy navigation.
- Incorporate subtle animations and transitions to enhance user engagement and provide visual feedback.
